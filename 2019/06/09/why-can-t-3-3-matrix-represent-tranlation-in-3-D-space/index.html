why can’t 3*3 matrix represent tranlation in 3-D space?

Usually we descibe rotation, scale and translation of a vector by 4*4 matrix, we call it homogeneous space, then project it to 3-D space by being devided by w. In this page we mainly talking about why should we should we do this.

Scale

Imaging there is a vector
$\vec x = \left [\begin{matrix}x_1 \\ x_2\\ x_3 \end{matrix} \right]$,
for scaling, we have to multiply each component of the vector by a factor.

Proportionally scaling

this means we scale up/down along the direction of the origin vector.

In the picture above we can see vector
$
\vec B =
\left [
\begin{matrix}
1\\2\\3
\end{matrix}
\right ]
,$ and $
\vec C =
\left [
\begin{matrix}
2\\4\\6
\end{matrix}
\right ]
$. Apparently we can find a matrix which makes this equation have a solution: $M·\vec B = \vec C$

$\begin{matrix}[2&0&0]\end{matrix}$ indicates that we take double $x_1$ from $\vec B$ and none of $x_2$ and $x_3$ to compose $x_1$ of $\vec C$.
$\begin{matrix}[0&2&0]\end{matrix}$ indicates that we take double $x_2$ from $\vec B$ and none of $x_1$ and $x_3$ to compose $x_2$ of $\vec C$.
$\begin{matrix}[0&0&2]\end{matrix}$ indicates that we take double $x_3$ from $\vec B$ and none of $x_1$ and $x_2$ to compose $x_3$ of $\vec C$.
thus we get $\vec C$.

Disproportionally scaling

this is a little bit different from proportionally scaling, but we can still use a 3*3 matrix to describe this operation.

similar to proportionally scaling, when we have $\vec M·\vec B = \vec C$, we got
$
M =
\left[
\begin{matrix}
1&0&0\\
0&2&0\\
0&0&1
\end{matrix}
\right]
$.
